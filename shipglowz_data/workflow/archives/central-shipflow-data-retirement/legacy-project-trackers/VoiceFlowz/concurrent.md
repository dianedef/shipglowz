# Concurrents et inspirations — VoiceFlowz

VoiceFlowz, sometimes written VoiceFlows, is the old name of the current WinFlowz app. This document is historical product-research context for voice, transcription, snippets, clipboard and Android overlay capabilities; it is not evidence of a separate current product.

## Lecture projet

The historical VoiceFlowz concept maps to the current WinFlowz app direction: voice, transcription, snippets, clipboard and Android overlay/keyboard capabilities. Les meilleurs liens restent voix, assistant personnel, mémoire et workspace.
