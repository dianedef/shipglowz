# Bug Record: BUG-2026-05-10-005

## Summary

- Title: Menu navigation ignores typed environment/folder name
- Current status: `in-diagnosis`
- Severity: `medium`
- Owner: unassigned
- Next command: `/sg-fix BUG-2026-05-10-005`

## Reproduction

- Preconditions:
  - Menu flow accessible via `M` then `N` in ShipFlow interactive CLI
  - Terminal environment supports interactive input (no web UI)
  - 2+ candidate items in prompt list
- Steps:
  1. Ouvrir le menu `Environments`, puis `Navigate`
  2. Dans la liste, taper le nom d'une entrée complète et appuyer sur Entrée
  3. Observer que le choix n'est pas pris et que le curseur semble avancer comme un Enter/Down

## Expected Behavior

- Un choix textuel complet doit être accepté lorsque la sélection est tapée à la main
- Le comportement doit rester correct pour les raccourcis à une touche (`a`, `b`, `n`, `x`, etc.)

## Observed Behavior

- Selon le rythme, la saisie texte complète peut être ignorée et la navigation se comporte de façon erratique.
