# Bug Record: BUG-2026-05-02-001

## Summary

- Title: Playwright MCP points to missing Google Chrome on Linux ARM64
- Current status: `closed`
- Severity: `medium`
- Owner: unassigned
- Next command: none

## Reproduction

- Preconditions:
  - Linux ARM64 environment.
  - Codex Playwright MCP configured without an explicit browser executable path.
  - Playwright-compatible Chromium binary exists under the user's Playwright cache.
- Steps:
  1. Invoke a Playwright MCP browser action.
  2. Let the MCP server initialize its browser.
  3. Observe the browser executable lookup.

## Expected Behavior

- Playwright MCP launches the local Playwright Chromium executable compatible with Linux ARM64.
- It must not require Google Chrome stable at `/opt/google/chrome/chrome`.

## Observed Behavior

- Playwright MCP tries to initialize the `chrome` distribution at `/opt/google/chrome/chrome`.
- The MCP reports `Run "npx playwright install chrome"`, but that Chrome channel is not the intended Linux ARM64 path.

## Evidence

- `mcp__playwright.browser_navigate` against `https://example.com` failed with: `Chromium distribution 'chrome' is not found at /opt/google/chrome/chrome`.
- `find ~/.cache/ms-playwright ...` found `/home/ubuntu/.cache/ms-playwright/chromium-1222/chrome-linux/chrome`.
