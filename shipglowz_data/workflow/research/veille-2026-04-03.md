## A CREUSER (priorite haute)

### #1 SEOJuice — Automatisation SEO + optimisation AI Search
**Lien :** https://seojuice.com/
- **Quoi** : SaaS qui automatise le SEO on-page (maillage interne, meta tags, schema, alt text, liens casses) via un snippet JS. Surtout : optimise pour les moteurs IA (ChatGPT, Claude, Gemini, Perplexity) — pas juste Google. Base proprietaire de 10 TB+. A partir de $9.99/mois (~$0.001/page).
- **Pour quel projet** : GoCharbon (8/10), Quit Coke (6/10)
- **Contenu** : N/A directement. Mais le blog SEOJuice produit du contenu sur l'AI Search Optimization (GEO) — tendance a suivre pour des articles GoCharbon sur le SEO en 2026.
- **Architecture** : Le snippet JS est compatible avec n'importe quel CMS y compris les sites Astro deployés statiquement. L'approche "zero-touch" (pas besoin de republier les pages) est interessante pour GoCharbon qui a ~290 posts. Le tracking des citations IA est un differenciateur fort — savoir quand Claude ou ChatGPT recommandent nos contenus.
- **Concurrence/inspi** : Leur positionnement GEO (Generative Engine Optimization) est en avance sur le marche. Les concurrents (Surfer, RankMath AI) ne font pas encore le tracking de citations dans les LLM. L'approche "open startup" (3067 sites, 589K pages) inspire confiance.
- **Opportunite collab** : Tester le tier gratuit (100 pages) sur GoCharbon immediatement. Si les resultats sont bons, deployer sur Quit Coke et les autres sites Astro.
- **Action concrete** :
  1. Creer un compte gratuit SEOJuice et connecter GoCharbon (100 pages gratuites = bon echantillon sur les 290)
  2. Mesurer l'impact sur le maillage interne et les meta tags pendant 2 semaines
  3. Analyser le rapport de citations IA — est-ce que les articles GoCharbon apparaissent dans ChatGPT/Claude ?

### #4 PainkillerIdeas — Validation d'idees startup par scraping de pain points
**Lien :** https://painkillerideas.com/
- **Quoi** : Plateforme IA qui scrape 150+ sources quotidiennement (Reddit, HN, GitHub, Product Hunt) pour identifier des vrais problemes utilisateurs, les scorer (intensite de douleur / taille marche / competition), et generer des rapports complets (TAM, concurrents, ICP, pricing, roadmap de validation). 526+ idees dans la base.
- **Pour quel projet** : GoCharbon (7/10), ContentFlowz (5/10)
- **Contenu** : Excellent materiau pour GoCharbon. Articles possibles : "Comment valider une idee startup en 5 minutes avec l'IA", "Painkillers vs Vitamines : la methode pour trouver des idees qui se vendent". Le concept "painkiller not vitamin" est un angle editorial fort pour le blog entrepreneur.
- **Architecture** : Le pipeline de scraping multi-source (Reddit + HN + GitHub + PH → scoring IA → rapport) est exactement le type d'architecture que ContentFlowz pourrait reproduire pour la veille de contenu. L'approche scoring/ranking avec willingness-to-pay est transposable.
- **Concurrence/inspi** : Leur methode de scoring (pain intensity 1-10, market size, competition) pourrait inspirer un systeme de scoring pour les idees de contenu dans ContentFlowz. Le modele "base d'idees curatable" est interessant.
- **Opportunite collab** : N/A
- **Action concrete** :
  1. Tester la plateforme avec 2-3 idees liees a nos projets (addiction recovery app, content automation, YouTube tools) pour voir la qualite des rapports
  2. Ecrire un article GoCharbon sur la methode "painkiller validation" avec un cas concret
  3. Etudier leur pipeline de scraping comme reference pour ContentFlowz

Features a comparer point par point : transcription multi-langue, chat IA, export agents, integrations (Slack, Zapier, Confluence). Le "Claras Academy" (tutoriels pour construire des knowledge bases) est un differenciateur contenu que TubeFlow pourrait repliquer.
